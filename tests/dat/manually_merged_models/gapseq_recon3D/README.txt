"M2_bacterial_model.xml" is a merged model of multiple bacteria, and it is base on the gapseq namespace. The gapseq namespace is an extended version of the SEED namespace.
"M1_recon3D_301_modified.xml" is the human metabolic model that can be found at https://github.com/VirtualMetabolicHuman/Recon/tree/master/Current_Version/Recon3D_301_Generic_Model, to which exchange reactions for ca, cl, zn ions and D-serine have been added.
